# Privacy Policy / プライバシーポリシー

## English

This tool stores only the data needed to operate posting and metrics retrieval on X, such as scheduled post text, scheduled times, tweet IDs, and public metrics (public_metrics). It also stores OAuth tokens required to access the X API (encrypted at rest on the machine where the tool runs).

Data is used only to operate the tool and is not sold or shared with third parties. Data is stored locally on the machine where the tool runs.

To stop data access by X, revoke the app in your X settings. To remove local data, delete the tool's local state files on the machine where it runs.

Contact: agentx9734@proton.me

## 日本語

本ツールは、Xでの投稿運用およびメトリクス取得に必要な情報のみ（例: 予約投稿の本文、投稿予定時刻、投稿ID、公開メトリクス public_metrics）を保存します。また、X APIにアクセスするために必要なOAuthトークンを保存します（ツール実行端末上で暗号化して保存します）。

保存したデータは本ツールの運用目的にのみ利用し、第三者へ販売・共有しません。データはツールの実行端末にローカル保存されます。

X側へのアクセスを停止するには、Xの設定から本アプリのアクセス権を取り消してください。ローカルデータを削除するには、ツール実行端末上の状態ファイルを削除してください。

連絡先: agentx9734@proton.me
